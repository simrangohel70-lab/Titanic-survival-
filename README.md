# 🚢 Titanic Survival Analysis using Python

## 📌 Project Overview

The **Titanic Survival Analysis** project is an Exploratory Data Analysis (EDA) project developed using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**.

The aim of this project is to analyze the Titanic dataset and identify the factors that influenced passenger survival using different visualizations and statistical analysis.

# 🎯 Objectives

- 📂 Load the Titanic dataset
- 🔍 Perform Exploratory Data Analysis (EDA)
- 📊 Visualize passenger data
- 🚹 Analyze survival by Gender
- 🎟️ Analyze survival by Passenger Class
- 👶 Study Age Distribution
- 💰 Study Fare Distribution
- 📈 Generate useful insights from the dataset

# 📈 Key Insights

🚺 Female passengers had a higher survival rate than male passengers.

🎟️ First Class passengers had the highest survival rate.

🚢 Third Class passengers had the lowest survival rate.

👶 Younger passengers had better chances of survival.

💰 Higher fare passengers generally had better survival chances.

📉 Missing values were mainly found in the Age and Cabin columns.
